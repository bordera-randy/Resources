### 1) DNS, IP, and Network Troubleshooting
- **MXToolbox** (DNS, MX, blacklist, SMTP, SPF/DKIM/DMARC checks): https://mxtoolbox.com/

- **DMARC Analyzer (report viewer)**: https://dmarcian.com/dmarc-inspector/
- **Google Admin Toolbox (Check MX, Messageheader, Dig)**: https://toolbox.googleapps.com/apps/main/

- **SSL Labs** (TLS scan): https://www.ssllabs.com/ssltest/
- **CRT.sh** (certificate transparency search): https://crt.sh/
- **SecurityHeaders** (HTTP security headers): https://securityheaders.com/
- **VirusTotal** (URL/file reputation): https://www.virustotal.com/

- **Webhook.site** (capture and inspect requests): https://webhook.site/
