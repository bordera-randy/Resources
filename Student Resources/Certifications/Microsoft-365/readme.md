## üí° Exam Tips
<a id="exam-tips"></a>

### General Strategies

1. **Read Carefully**: Microsoft exams test detailed knowledge; read every word of questions and answers
2. **Scenario-Based**: Most questions are scenario-based; understand practical applications
3. **Plan Your Time**: 
   - MS-900: ~1.5 minutes per question
   - MS-101/102/702: ~2 minutes per question
4. **Skip and Return**: Mark difficult questions and return if time permits
5. **Review Before Submitting**: Check your answers before final submission

### Study Approach

1. **Week 1-2**: Review official Microsoft Learn modules
2. **Week 3-4**: Deep dive into each exam objective
3. **Week 5-6**: Hands-on practice in test environments
4. **Week 7**: Practice exams and weak area review
5. **Final Review**: Final practice exam and exam day prep

### Common Question Patterns

- **Policy Configuration**: Which setting achieves this goal?
- **Troubleshooting**: What's the cause of this problem?
- **Architecture**: How should this be designed?
- **Service Integration**: How do services work together?
- **Compliance**: What feature meets this requirement?
