## ðŸ§ª Practice Questions

### Question 1: User Management
**You need to provision 50 new employees at once. Which method is most efficient?**

A) Create each user individually in the admin center  
B) Use bulk user creation via CSV import  
C) Have each employee self-register  
D) Create a template user and duplicate  

**Answer: B) Use bulk user creation via CSV import**  
*Bulk operations with CSV files are the most efficient method for creating multiple users at once.*

### Question 2: Licensing
**A user has a Microsoft 365 Business Premium license but cannot access Microsoft Project Online. What is the likely cause?**

A) License not assigned properly  
B) Project Online is not included in Business Premium  
C) User's account is disabled  
D) Office desktop apps not installed  

**Answer: B) Project Online is not included in Business Premium**  
*Business Premium includes core M365 apps but not Project Online; that requires a separate subscription.*

### Question 3: Teams Administration
**You need to prevent external users from joining Teams meetings in your organization. Where should this policy be configured?**

A) SharePoint admin center  
B) Exchange admin center  
C) Teams admin center  
D) Microsoft 365 admin center  

**Answer: C) Teams admin center**  
*Teams admin center contains guest access and meeting policies specific to Teams.*

### Question 4: SharePoint Administration
**A user reports they cannot access a SharePoint site their manager created. What is the most common reason?**

A) SharePoint is down  
B) Site permissions not configured  
C) User's license revoked  
D) Browser cache issue  

**Answer: B) Site permissions not configured**  
*SharePoint sites require explicit permissions; by default, only creators have access.*

### Question 5: Password Policy
**Your organization requires passwords to be changed every 90 days. Where is this policy configured?**

A) Azure AD/Entra ID  
B) Exchange admin center  
C) Teams admin center  
D) Microsoft 365 admin center  

**Answer: A) Azure AD/Entra ID**  
*Password policies are managed in Azure AD under Security > Password policy.*
