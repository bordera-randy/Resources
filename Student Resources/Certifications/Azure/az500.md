## üéØ Tips & Study Strategy

**Proven strategies to master AZ-500:**

1. **Hands-on with core services:**
   - Deploy and configure Azure Firewall, NSGs, and Azure Policy
   - Create Conditional Access policies and test MFA scenarios
   - Explore Defender for Cloud recommendations and secure score
   - Query logs in Microsoft Sentinel using KQL

2. **Understand Zero Trust principles:**
   - Scenario questions expect understanding of layered security
   - Know how to implement defense-in-depth across identity, network, data
   - Review Microsoft's Zero Trust architecture framework

3. **Master RBAC and PIM:**
   - Know the difference between Azure AD roles and Azure roles
   - Understand role activation, approval workflows, and just-in-time access
   - Practice custom role creation and scope assignment

4. **Deep dive into Conditional Access:**
   - Memorize common policies: MFA, device compliance, risk-based access
   - Understand how CA policies combine conditions and controls
   - Know exclusion and inclusion logic

5. **Study Microsoft Security Baselines:**
   - Review IaaS, PaaS, and SaaS security baselines
   - Understand specific hardening recommendations per service
   - Know which resources apply to exam scenarios

6. **Practice with Defender for Cloud:**
   - Simulate security alerts and understand remediation steps
   - Know how to interpret secure score and recommendations
   - Understand Just-In-Time VM access and adaptive network hardening

7. **Learn Key Vault and encryption:**
   - Understand data at rest and data in transit encryption
   - Know Key Vault access policies, RBAC, and managed identities
   - Practice secret and key rotation strategies

8. **Mock exams under timed conditions:**
   - Use 100‚Äì120 minute practice exams to simulate the real exam
   - Review every incorrect answer against official documentation
   - Focus on scenario-based questions which are heavily weighted

**Recommended study timeline:**
- **Beginner (no Azure security background):** 10‚Äì14 weeks (10‚Äì15 hours/week)
- **Some Azure experience:** 6‚Äì8 weeks (10‚Äì15 hours/week)
- **Azure admins / security professionals:** 3‚Äì4 weeks (focused review and practice)
