## ðŸ§ª Practice Questions

### Question 1: Cloud Models
**Which cloud service model provides the highest level of management responsibility for the customer?**

A) SaaS  
B) PaaS  
C) IaaS  
D) Hybrid Cloud  

**Answer: C) IaaS**  
*With IaaS, customers manage the operating system, middleware, and applications, while the provider manages infrastructure.*

### Question 2: Microsoft 365 Services
**Your organization needs a solution for team collaboration, file sharing, and video conferencing. Which Microsoft 365 service should be the primary hub?**

A) OneDrive  
B) SharePoint  
C) Teams  
D) Outlook  

**Answer: C) Teams**  
*Teams serves as the central hub for chat, meetings, calls, and integrates with OneDrive and SharePoint.*

### Question 3: Security Feature
**What is the primary benefit of enabling Multi-Factor Authentication (MFA) in Microsoft 365?**

A) Reduces licensing costs  
B) Increases storage capacity  
C) Adds an additional layer of identity verification  
D) Improves internet speed  

**Answer: C) Adds an additional layer of identity verification**  
*MFA requires users to provide multiple forms of verification, significantly reducing unauthorized access.*

### Question 4: Compliance
**Which Microsoft 365 feature helps organizations retain or delete emails and documents based on specific policies?**

A) Audit logs  
B) Retention policies  
C) Data loss prevention  
D) Conditional access  

**Answer: B) Retention policies**  
*Retention policies automate how long records are kept before deletion, supporting compliance requirements.*

### Question 5: Pricing Model
**What is the primary licensing model for Microsoft 365 business plans?**

A) Device-based licensing  
B) Per-user subscription  
C) One-time purchase  
D) Volume discount only  

**Answer: B) Per-user subscription**  
*Microsoft 365 uses per-user, subscription-based licensing across all its business plans.*
