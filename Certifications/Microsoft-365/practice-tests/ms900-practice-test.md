### Question 1
**Which cloud service model provides the customer with the most responsibility for managing the infrastructure?**

A) Software as a Service (SaaS)  
B) Platform as a Service (PaaS)  
C) Infrastructure as a Service (IaaS)  
D) Hybrid Cloud  

<details>
<summary>Answer</summary>
<b>C) Infrastructure as a Service (IaaS)</b>

With IaaS, the customer manages the operating system, middleware, applications, and data, while the provider manages the infrastructure (servers, networking, virtualization). PaaS requires less customer management (no OS), and SaaS requires almost none (just data and access configuration).
</details>

C) Makes the customer fully responsible for security  
D) Makes the provider fully responsible for security  

<details>
<summary>Answer</summary>

### Question 6
