## ðŸ§ª Practice Questions

### Question 1: Device Management
**You need to ensure that all company devices are encrypted and have antivirus protection enabled before accessing Microsoft 365. Which Microsoft 365 feature should you implement?**

A) Device enrollment only  
B) Compliance policies  
C) Conditional access  
D) Intune app protection  

**Answer: C) Conditional access**  
*Conditional access allows you to block or grant access based on device compliance status, requiring encryption and antivirus as conditions.*

### Question 2: Threat Protection
**Which Microsoft 365 threat protection service is specifically designed to detect and respond to advanced threats on devices?**

A) Defender for Office 365  
B) Defender for Endpoint  
C) Defender for Identity  
D) Cloud App Security  

**Answer: B) Defender for Endpoint**  
*Defender for Endpoint provides endpoint detection and response (EDR) capabilities for devices.*

### Question 3: Data Protection
**Your organization needs to prevent employees from copying sensitive customer data to personal cloud storage services. Which feature should be configured?**

A) Sensitivity labels  
B) Information Rights Management  
C) Data loss prevention (DLP)  
D) eDiscovery hold  

**Answer: C) Data loss prevention (DLP)**  
*DLP rules can detect and prevent sensitive data from being shared to unauthorized locations.*

### Question 4: Identity Management
**An employee requests access to a sensitive resource that requires approval from their manager. Which Azure AD feature enables this workflow?**

A) Self-service password reset  
B) Privileged Identity Management (PIM)  
C) Access reviews  
D) Conditional access  

**Answer: B) Privileged Identity Management (PIM)**  
*PIM enables time-limited privileged access with approval workflows.*

### Question 5: Compliance
**Which Microsoft 365 feature automatically retains emails based on organizational policies and prevents deletion?**

A) Litigation hold  
B) Retention policies  
C) DLP policies  
D) Audit logs  

**Answer: B) Retention policies**  
*Retention policies automatically preserve and manage content based on defined rules.*
